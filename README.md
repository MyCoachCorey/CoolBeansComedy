# Cool Beans Comedy — Pages

Working drafts, briefs, and shareable docs for Cool Beans Comedy.

Deployed via Cloudflare Pages → live URLs for sharing instead of attaching HTML files.

## Files

- `index.html` — landing page that lists everything
- `teams-vs-agents.html` — comparison brief

## How to add a new doc

1. Drop a new `.html` file in this folder
2. Add a link to it in `index.html`
3. `git add . && git commit -m "Add <name>" && git push`
4. Cloudflare Pages auto-deploys — live URL within a minute
